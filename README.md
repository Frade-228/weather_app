# Прогноз опадів на основі даних Open-Meteo

## Мета проєкту

Мета цього проєкту — створити міні-сервіс прогнозування опадів на основі реальних метеоданих, отриманих через API Open-Meteo.

У проєкті:
- завантажуються щоденні погодні дані через HTTP-запит;
- формується датасет для машинного навчання;
- створюється цільова змінна для класифікації: будуть опади чи ні;
- навчається модель класифікації;
- обчислюються метрики якості моделі;
- виконується прогноз опадів для наступного дня;
- результат відображається у Streamlit-застосунку.

## Опис ідеї проєкту

Проєкт використовує відкритий сервіс Open-Meteo для отримання щоденних погодних даних за вказаними координатами.

Основна ідея полягає в тому, щоб на основі погодних характеристик поточного дня передбачити, чи будуть опади наступного дня.

Як цільову змінну використано:
- `0` — опадів немає;
- `1` — опади є.

Цільова змінна формується за правилом:
- якщо `precipitation_sum > 0`, то клас = `1`;
- якщо `precipitation_sum = 0`, то клас = `0`.

У якості моделі класифікації використано `RandomForestClassifier`.

## Використані технології

У проєкті використані такі бібліотеки:
- `requests` — для HTTP-запиту до Open-Meteo;
- `pandas` — для обробки табличних даних;
- `scikit-learn` — для побудови моделі машинного навчання;
- `streamlit` — для створення простого веб-інтерфейсу.
